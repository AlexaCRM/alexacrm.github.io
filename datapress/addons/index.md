DataPress Premium unlocks advanced functionality that goes far beyond basic Dataverse integration.
With premium plans, you can build fully interactive, personalized, secure, data‑driven WordPress experiences powered directly by Dataverse. These capabilities enable real business scenarios such as authenticated portals, dynamic content for each user, user profile synchronization, multisite environments, and automated record binding.
The table below compares the feature sets across the DataPress Builder, DataPress Portal, and DataPress Unlimited plans.

| Forms (Power Apps, Gravity Forms)         | ✔️                | ✔️               | ✔️                   |
| Elementor                                 | ✔️                | ✔️               | ✔️                   |
| Custom fields                             | ✔️                | ✔️               | ✔️                   |
| Free Dataverse Dev instances              | ✔️                | ✔️               | ✔️                   |
